# Todo App

A task management application developed using Flutter and Firebase to help users manage daily tasks efficiently.

The app allows users to create, update, complete, and delete tasks with a clean and user-friendly mobile interface.

---

## 🚀 Features

- User-friendly interface
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Firebase database integration
- Realtime task updates
- Mobile responsive design

---

## 🛠 Technologies Used

- Flutter
- Dart
- Firebase
- Cloud Firestore

---


## ▶️ How To Run

1. Clone the repository

```bash
git clone https://github.com/vinitsonii/TODOAPP.git
```

2. Open the project in Android Studio or VS Code

3. Install dependencies

```bash
flutter pub get
```

4. Configure Firebase

- Add `google-services.json`
- Enable Firebase services

5. Run the application

```bash
flutter run
```
